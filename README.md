# Nova Java Spring Boot API Standard

Libreria pura Java para estandarizar contratos HTTP en APIs REST de Spring Boot.

## Instalacion

```gradle
dependencies {
    implementation("pe.edu.nova.java.libs:nova-api-standard:0.1.0")
}
```

## Configuracion

Anotar controllers con `@ApiStandard` y devolver `ApiResponse<T>` para formato uniforme.
